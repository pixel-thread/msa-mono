Subscription Layout Options

How should we organize the active plan, user details, and payment history?

A

Unified Single-Scroll

A single long page with clear sections:

  • Top: Profile Card (Name, ID, Email)
  • Middle: Billing Stats Grid (Total Paid, Due, etc.)
  • Center: Active Plan card with "Pay" button
  • Bottom: Payment History list (expandable)

Best for a quick overview of everything in one place.

B

Tabbed Layout

Split view for better focus:

  • Tab 1 (Plan): Profile summary + Active Plan + Pay button
  • Tab 2 (History): Detailed Billing Stats + Full Audit Trail list

Best for managing long payment histories without cluttering the main view.

Audit Trail Detail Level

The history will include: Date, Amount, Status, Currency, Created By, and Allocation details (which month/period the payment covered).
